# Plastic-Free Awareness Movement

## 📌 Project Description
This project focuses on spreading awareness about plastic pollution and encouraging people to adopt eco-friendly alternatives. It includes survey analysis, awareness activities, and a web-based chatbot system.

## 🚀 Features
- Awareness campaign
- Survey of residents
- Cloth bag promotion
- AI chatbot for plastic awareness

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
Download the project and open index.html in any browser.
